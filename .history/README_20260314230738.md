# 🌄 WallScape

**WallScape** is a collection of high-quality wallpapers gathered in one place.
The goal of this repository is to make it easy to browse, preview, and download beautiful wallpapers without complicated folder structures.

---

## ✨ Features

* 🖼️ High-resolution wallpapers
* ⚡ Simple and clean repository structure
* 📥 Easy to download
* 💻 Works great for desktop, laptop, and mobile backgrounds

---

## 📂 Repository Structure

```
WallScape/
│
├── wallpaper1.jpg
├── wallpaper2.png
├── wallpaper3.jpg
├── wallpaper4.png
└── ...
```

All wallpapers are stored directly in the repository for quick access.

---

## 📥 Downloading Wallpapers

**Download a single wallpaper**

1. Open the image file.
2. Click **Download** or **Save image**.

**Download the full collection**

1. Click the **Code** button.
2. Select **Download ZIP**.
